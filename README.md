# Тренажер «Р» / „R“ Tréner / “R” Trainer

Односторінковий вебзастосунок для постановки звуку **«Р»** — 56-денний курс із трьох фаз
(гімнастика → вібрація → автоматизація). Працює офлайн, прогрес зберігається у браузері
(`localStorage`). Три мови інтерфейсу та вправ: 🇺🇦 українська, 🇸🇰 slovenčina, 🇬🇧 English.

**▶ Демо:** https://wizartclub-cyber.github.io/r-trainer/

A single-file web app that guides a 56-day course to produce the rolled **R** sound.
Three phases (gymnastics → vibration → automation), per-day checklist with rep counters and
timers, a progress calendar, and a daily vibration log. No build step, no backend — one HTML file.

## Запуск / Run

Просто відкрийте `index.html` у браузері, або відкрийте демо-посилання вище.
Нічого встановлювати не треба.

Just open `index.html` in any browser (or the demo link above). Nothing to install.

## Структура / Structure

- `index.html` — увесь застосунок: розмітка, стилі й логіка в одному файлі.
  Контент кожної мови лежить в об'єкті `I18N` (рядки інтерфейсу + вправи `EX`).
  Ідентифікатори вправ спільні для всіх мов, тому прогрес не губиться при зміні мови.

## Зроблено / Done

- ✅ **PWA + офлайн** — `manifest.json` + service worker: встановлюється на телефон, працює без інтернету.
- ✅ **Експорт/імпорт прогресу** — кнопки в налаштуваннях: зберегти прогрес у JSON-файл і завантажити назад (бекап / передати логопеду).
- ✅ **Три мови** — 🇺🇦 / 🇸🇰 / 🇬🇧, структура `I18N` готова до додавання нових.
- ✅ **Запис голосу** — `MediaRecorder`: дитина записує спробу, слухає, завантажує файл (для логопеда). Запис у межах сесії, не зберігається між перезавантаженнями.

## Далі / Ideas to grow it

1. **Постійне зберігання записів** — IndexedDB: тримати останній запис на кожен день, щоб чути прогрес голосу. (Зараз запис лише в сесії.)
2. **Аудіо-зразки** — короткі записи правильного «Р» і кожної вправи (звук важливіший за текст для дитини). *Потрібні реальні записи.*
3. **Нагадування** — щоденний пуш. Локальні нотифікації обмежені; для справжніх пушів потрібен бекенд.
4. **Гейміфікація** — бейджі, серії, маленькі винагороди за виконані дні.
5. **Режим для батьків/логопеда** — окремий екран із нотатками, графіком вібрації за тижнями, коментарями фахівця.
6. **Гнучкий курс** — редагування вправ, доз і тривалості фаз під конкретну дитину.
7. **Більше звуків** — тренажери для Л, шиплячих тощо (та сама структура).
8. **Синхронізація** — за потреби бекенд/хмара, щоб прогрес був на кількох пристроях.

*Порядок — приблизно за співвідношенням користь/зусилля.*
