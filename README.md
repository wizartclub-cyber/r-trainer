# Тренажер «Р» / „R“ Tréner / “R” Trainer

Односторінковий вебзастосунок для постановки звуку **«Р»** — 56-денний курс із трьох фаз
(гімнастика → вібрація → автоматизація). Працює офлайн, прогрес зберігається у браузері
(`localStorage`). Три мови інтерфейсу та вправ: 🇺🇦 українська, 🇸🇰 slovenčina, 🇬🇧 English.

**▶ Демо:** https://wizartclub-cyber.github.io/r-trainer/

A single-file web app that guides a 56-day course to produce the rolled **R** sound.
Three phases (gymnastics → vibration → automation), per-day checklist with rep counters and
timers, a progress calendar, and a daily vibration log. No build step, no backend — one HTML file.

## Запуск / Run

Просто відкрийте `index.html` у браузері, або відкрийте демо-посилання вище.
Нічого встановлювати не треба.

Just open `index.html` in any browser (or the demo link above). Nothing to install.

## Структура / Structure

- `index.html` — увесь застосунок: розмітка, стилі й логіка в одному файлі.
  Контент кожної мови лежить в об'єкті `I18N` (рядки інтерфейсу + вправи `EX`).
  Ідентифікатори вправ спільні для всіх мов, тому прогрес не губиться при зміні мови.

## Як розвивати проєкт / Ideas to grow it

1. **Нагадування** — щоденний пуш через the Notifications API / PWA, щоб не пропускати заняття.
2. **PWA + офлайн** — `manifest.json` + service worker: встановлення на телефон, іконка, повний офлайн.
3. **Експорт/імпорт прогресу** — кнопка «зберегти у файл / поділитися з логопедом» (JSON), бо `localStorage` живе лише в одному браузері.
4. **Аудіо-зразки** — короткі записи правильного «Р» і кожної вправи (звук важливіший за текст для дитини).
5. **Запис голосу** — `MediaRecorder`: дитина записує спробу, порівнює із зразком; логопед слухає пізніше.
6. **Гейміфікація** — бейджі, серії, маленькі винагороди дитині за виконані дні.
7. **Режим для батьків/логопеда** — окремий екран із нотатками, графіком вібрації за тижнями, коментарями фахівця.
8. **Гнучкий курс** — редагування вправ, доз і тривалості фаз під конкретну дитину.
9. **Більше мов і звуків** — структура `I18N` вже готова; так само можна додати тренажери для інших звуків (Л, шиплячі).
10. **Синхронізація** — за потреби бекенд/хмара, щоб прогрес був на кількох пристроях.

*Порядок — приблизно за співвідношенням користь/зусилля. Почав би з PWA + нагадувань (3–4) та аудіо-зразків.*
